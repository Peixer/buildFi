# Investor Dashboard

## Purpose
The Investor Dashboard gives investors a single place to understand exposure, project status, returns, and cash flow across their positions.

## Key Questions It Must Answer
- Which projects have I funded?
- What is my current exposure?
- How much capital is locked vs released?
- Which projects are progressing well?
- Which milestones are pending or delayed?
- What returns or payouts should I expect?

## Core Sections

### Portfolio Summary
- Total invested capital
- Locked capital
- Released capital
- Number of active projects
- Portfolio-level ROI or projected return

### Project Positions
For each project:
- Project name
- Builder
- Amount invested
- Ownership or debt position
- Current milestone
- Funding state
- Health indicator
- Expected payout window

### Activity Feed
- New project updates
- Milestone approvals
- Payment releases
- Builder communications
- Risk alerts

## MVP Scope
- Portfolio summary cards
- List of invested projects
- Project status and funding state
- Recent activity timeline
- Click-through to project details

## UX Priorities
- Instant comprehension
- Financial and operational state shown together
- Important changes surfaced first
- Trust and performance visible without extra digging

## Why This Matters
Investor confidence is not built only at the moment of investment. It is maintained through visibility after the capital is committed.

## Future Extensions
- Cross-project analytics
- Cash flow forecasting
- Secondary liquidity positions
- Notification center
- Builder comparison tools

## Demo Narrative
After funding a project, the investor sees it appear in the dashboard. Once a milestone is approved, locked capital decreases, released capital increases, and the latest project event appears in the activity feed.
